# Аудио и видео файлы
audio - позволяет вставить аудиофайл. Атрибуты: controls - отображает стандартные элементы управления (play, пауза, громкость), autoplay - автоматическое воспроизведение, loop - зацикливание, muted - отключение звука по умолчанию.  

video - позволяет вставить видео. Атрибуты: width/height - размеры плеера, poster - изображение-заставка до воспроизведения,preload - загрузка видео заранее.
