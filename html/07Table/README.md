# Таблицы
table - создание таблицы  
tr - строка таблицы  
td - ячейка с данными  
th - заголовок таблицы  

примеры таблиц  

caption - объединенный заголовок  

Семантические элементы:  
thead - шапка таблицы  
tbody — основное тело таблицы  
tfoot — подвал  

td colspan="2" объединяет строки в одну
